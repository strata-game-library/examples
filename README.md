## Quick Start

```bash
git clone https://github.com/jbcom/nodejs-strata-examples.git
cd nodejs-strata-examples/basic-terrain
pnpm install
pnpm dev
```

## Dependencies

All examples use the published npm package:

```json
{
  "dependencies": {
    "@jbcom/strata": "^1.4.0"
  }
}
```
